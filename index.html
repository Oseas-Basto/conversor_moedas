Perguntas Frequentes

De onde vêm as cotações?

As cotações são obtidas através da AwesomeAPI, uma API pública de dados financeiros.

As cotações são em tempo real?

As informações são atualizadas periodicamente conforme a disponibilidade da API.

O conversor é gratuito?

Sim. O uso do ConversorMoedas é gratuito.

Posso usar os dados para operações financeiras?

Os dados são apenas informativos. Antes de realizar operações financeiras, consulte bancos, corretoras ou instituições autorizadas.
